## üåü Features

- **12 Waste Categories**: Battery, Biological, Brown Glass, Cardboard, Clothes, Green Glass, Metal, Paper, Plastic, Shoes, Trash, White Glass
- **Deep Learning**: EfficientNet-B0 architecture for accurate classification
- **Real-time Prediction**: Instant results with confidence scores
- **Recycling Guidance**: Specific disposal instructions for each waste type
- **Environmental Tips**: Educational facts about waste impact
- **Camera Support**: Take photos directly in the app
